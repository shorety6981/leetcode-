# 两数之和
## 第一次通过的代码
```
def twoSum(self, nums, target):
  for i in range(len(nums)):
    a = nums[i]
    for j in range(len(nums)):
      b = nums[j]
      if i<j and a+b==target:
         return [i,j]
```
虽然通过了，但时间长达5340ms,需要继续精简
### 如何精简
循环嵌套的时间复杂度较高，降低循环的使用率
